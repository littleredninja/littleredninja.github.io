Let's get Loopy

While, until, loop, for, times and each

Friday, June 6, 2014

Lots of times in programs, you'll need code to perform a block of code over and over. This can be accomplished in several different ways. We'll look over some simple examples of while, until, loop, for, times and each.

While, Until and Loop

While, until and loop perform actions until a condition is met. These are good for instances when you don't know how many times you'll have to loop. But look out, because you'll need to make sure that at some point the loop's conditions can be met, otherwise you'll end up with an infinite loop, which will cause you're program to crash.

number = 1
while number < 5
  puts number
  number += 1
end

The above example will puts the number until 4, at which point it will stop because number will no longer be < 5.

number = 1
until number > 5
  puts number
  number += 1
end

The until loops works exactly like the while loop, but it needs the opposite condition. It will print 1 through 5, because it will not evaluate to false until the number gets to 6.

The loop method defines its conditional by saying "break if" then naming the condition.

number = 1
loop do
  number += 1
  puts number
  break if number >= 5
end

For and Times

For and times loops are useful for instances when you know how many times you'll need to perform a specific action. For also lets you include a range of numbers, so you can start and stop when you'd like.

for var in 1..5
  puts var
end

Times is a built-in Ruby method that lets you perform a block a definite number of times. It looks like this:

5.times { puts "Hello, world!" }

Each

The each method allows you to perform an action on each item in a collection. Since it can be used on arrays and hashes, this makes it very powerful for looping through elements in those objects.

array.each { |x| puts x }
