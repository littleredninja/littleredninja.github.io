The Display Property

I just spent a million hours trying to make a picture center in a div, and the one thing that made it happen was the display property in my css code.

So what does that display property do anyway? What is the difference between inline and inline-block?

Display property has four popular options: inline, inline-block, block and none.

Inline + Block = Inline Block

Inline elements are elements that will not create new row in a layout. They cannot have a defined height and width in css, and will accept margins on the left and right, but not on the top and bottom. Elements like em, anchor, span and img are inline by default.

These are inline divs.

Block elements create a new row in a layout. They can accept a defined height and width, and margins can be set on all sides. Elements like p, h1, div, header and footer are block elements by default.

These divs have a declared height and width of 150px.
Each div creates a new row.
These are block divs.

But if you'd like to make divs that sit side by side, or set a fixed width or height for a inline element, you can declare a display property of inline block. This allows the flexibility of defining a block element without creating a new row.

But they are able to sit side by side.
These are inline-block divs.

None means none.

None is pretty self explanatory, as it just means that the item will not show up in your page.
